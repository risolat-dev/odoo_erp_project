# Odoo Modullari va Funksiyalari 

| Modul nomi | Vazifasi va nima uchun kerak?                                                                              |
| :--- |:-----------------------------------------------------------------------------------------------------------|
| **Sales** | Mijozlar buyurtmalarini shakllantirish va sotuv tarixini yuritish uchun.                                   |
| **Purchase** | Xitoydan keladigan import tovarlarni xarid qilish va tannarxni hisoblash uchun.                  |
| **Inventory** | Ombor qoldiqlarini aniq nazorat qilish va harakatlarni kuzatish uchun.                           |
| **Accounting** | To'lovlarni qabul qilish, invoice-larni yopish va 1C o'rniga moliya hisobini yuritish uchun. |
| **Approvals** | Katta summadagi buyurtmalarni menejer tomonidan tasdiqlash uchun.                                |
| **Custom Modules** | Mijoz qarz limitini tekshirish va maxsus tasdiqlash mantiqi uchun.                           |
